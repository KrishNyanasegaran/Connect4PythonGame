# Connect4PythonGame

Developed an interactive Connect 4 game using Python, implementing both Player vs Player and Player vs AI modes with a graphical user interface.

• Designed and built the full game logic including turn-based gameplay, win detection, and board management
• Implemented a GUI using Tkinter, including a main menu, interactive board, and real-time status updates
• Integrated decision-making AI logic for single-player mode, allowing the computer opponent to select moves strategically
• Added features such as leaderboard tracking, game reset, and menu navigation

This project strengthened my understanding of algorithms, event-driven programming, decision-making logic, and user interface design.
